# harmonic-compliance-forecast
This repository contains the full pipeline for a telemetry-first probabilistic harmonic compliance risk forecasting framework, developed as part of an undergraduate thesis at the University of Wollongong in Dubai, supervised by Dr. Mohamad Nassereddine, with funding from the DEWA Al Baheth Programme. The framework forecasts voltage total harmonic distortion at low-voltage points of common coupling using only IEC 61000-4-30 aggregated indices, without requiring waveform-level instrumentation. A multi-output histogram gradient boosting regressor produces point forecasts across multiple horizons, wrapped in adaptive split conformal prediction intervals for distribution-free uncertainty quantification. The model is benchmarked against five baselines and validated through zero-shot frozen-model transfer across three independently instrumented sites. Related work has been published at IMCET 2026 and is currently under review at SUSTAIN KSA and the IEEE Open Access Journal of Power and Energy.
